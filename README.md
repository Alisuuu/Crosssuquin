Como usar os Addons
Atenção: O app não cria ícone no menu. Ele funciona direto pelos botões da barra de notificações (configurações rápidas).
1. Adicionar o botão
Baixe a barra de notificações e clique em Editar (lápis).
Procure pelo nome do addon que você baixou na lista.
Nota: Eles funcionam separadamente. Se você baixou mais de um addon, vai ter um botão diferente para cada um.
Arraste o botão para a área visível (parte de cima).
2. Liberar o uso
Toque no botão que acabou de adicionar. Ele vai pedir permissão de Sobreposição. Ative essa permissão e volte.
3. Usando
É só tocar no botão (toggle) na barra de notificações.
Tocou: O addon aparece na tela.
Tocou de novo: Ele some.
