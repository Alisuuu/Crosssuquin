# Suquin Addons

> ** Importante:** Este aplicativo **não possui ícone na tela inicial**. Ele funciona exclusivamente através das **Configurações Rápidas** (aquela barra onde fica o Wi-Fi e Bluetooth).

Cada addon funciona de forma independente. Siga os passos abaixo para ativar o que você baixou.

##  Configuração Inicial

Como o app não abre da forma tradicional, você precisa adicionar o atalho manualmente na primeira vez.

### 1. Adicione o Toggle (Botão)
1. Deslize a barra de notificações totalmente para baixo.
2. Toque no ícone de **Lápis** (Editar) para organizar os botões.
3. Procure pelo **nome específico do addon** que você baixou na lista de botões inativos (geralmente no final).
4. Segure e arraste o botão para a área ativa (parte de cima).

### 2. Conceda a Permissão
Para que o addon desenhe na tela, ele precisa de permissão de sobreposição.

1. Toque no botão do addon que você acabou de adicionar na barra.
2. O sistema abrirá uma tela de permissão automaticamente.
3. Ative a opção **"Permitir sobreposição a outros apps"**.
4. Volte para a tela inicial.

##  Como Usar

Agora está tudo pronto. O funcionamento é simples:

- **Toque no botão:** Ativa o addon (visível na tela).
- **Toque novamente:** Desativa/Remove da tela.

---

### Solução de problemas
* **Não achei o botão:** Certifique-se de procurar pelo *nome do addon* específico na lista de edição das configurações rápidas.
* **O botão não funciona:** Verifique se a permissão de sobreposição foi concedida corretamente nas configurações do Android.
