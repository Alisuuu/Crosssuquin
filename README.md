> **Aviso:** Este aplicativo **não possui ícone na tela inicial**.
> O CrossS funciona **exclusivamente através das Configurações Rápidas** (barra de notificações).

O **CrossS** é uma ferramenta direta, leve e focada em desempenho, desenvolvida para auxiliar na gameplay sem causar queda de FPS ou instabilidade no sistema.

---

## Características Principais

* Sem interface gráfica
* Nenhuma configuração manual
* Consumo mínimo de recursos
* Ativação rápida por botões (toggles)
* Possibilidade de usar múltiplos módulos simultaneamente

---

## Instalação e Configuração

Como o aplicativo não possui interface, todo o processo é feito pelas Configurações Rápidas.

### 1. Adicionar os Botões

1. Deslize a barra de notificações totalmente para baixo
2. Toque no ícone de **Editar** (lápis)
3. Arraste os botões do CrossS para a área ativa
4. Repita o processo para cada módulo disponível

---

### 2. Conceder Permissões

1. Toque em qualquer botão do CrossS
2. O sistema solicitará a permissão **"Sobrepor a outros apps"**
3. Ative a permissão e retorne

---

## Como Usar

* Para ativar um módulo: toque no botão correspondente
* Para desativar: toque novamente
* É possível usar vários módulos ao mesmo tempo

---

## Compatibilidade

* Android 12 ou superior
* versão do shizuku que eu recomendo, a função cão de guarda te faz instalar e esquecer: https://github.com/thedjchi/Shizuku/releases
---

## Desinstalação

* Pressione e segure qualquer botão do CrossS nas Configurações Rápidas
  **ou**
* Acesse Configurações > Apps e desinstale normalmente
