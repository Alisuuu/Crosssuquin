# Suquin Addons

> ** Aviso:** Este app **não possuei ícone na tela inicial**. Ele funcionam exclusivamente através das **Configurações Rápidas** (Barra de Notificações).

o app CrossS é uma ferramenta direta e leve para auxiliar na sua gameplay. Ele foi desenvolvido para ser **simples** e de **alto desempenho**.

## Características Principais

* **Zero Configuração:** Não existem menus ou ajustes. o app já vem calibrado para oferecer o melhor desempenho possível sem pesar no jogo.
* **Uso Simultâneo:** Você pode ativar varias retículas diferentes ao mesmo tempo, bastando ativar os botões correspondentes.

##  Instalação e Configuração

Como não há interface do aplicativo, o processo é feito pela barra de status:

### 1. Adicione os Botões (Toggles)
1.  Deslize a barra de notificações totalmente para baixo.
2.  Toque no ícone de **Lápis** (Editar) para organizar os botões.
4.  Arraste o botão para a área ativa (parte de cima).
    * *Repita isso para cada addon que você tiver instalado.*

### 2. Conceda a Permissão
Para funcionar sobre o jogo, o Android exige permissão.

1.  Toque no botão do addon recém-adicionado.
2.  O sistema pedirá a permissão de **"Sobrepor a outros apps"**.
3.  Ative a chave e volte.
##  Como Usar

Com os botões na barra, é só jogar:

* **Para Ativar:** Toque no botão da retícula desejada na barra de notificações.
* **Para Desativar:** Toque novamente para remover.
* **Combinar:** Quer usar dois ou três reticulas? É só ativar todos os botões correspondentes na barra.

---

### Solução de Problemas
* **O botão não faz nada:** Verifique se a permissão de sobreposição foi concedida.
* **Não acho opções de ajuste:** Isso é intencional. O app é feito para garantir que não haja queda de FPS ou lag durante o uso.
